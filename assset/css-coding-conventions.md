
## BEM
### B for Block
```css
    .block {

    }
```
```html
    <div class="block"></div>
```
### E for Elements
```css 
    .sticky-man {

    }

    .sticky-man__head {

    }

    .sticky-man__foot {
        
    }
```
```html
 <div class="sticky-man">
    <div class="sticky-man__head">

    </div>
    <div class="sticky-man__foot">

    </div>
 </div>
```
### M for Modifiers
```css
    .stick-man--blue {

    }

    .sticky-man__head--small {

    }
```
```html
<div class="sticky-man__head">
    <div class="sticky-man__head--small">
    
    </div>

</div>
```
